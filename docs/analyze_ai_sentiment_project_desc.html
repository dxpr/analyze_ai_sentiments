The AI Sentiment Analysis is part of the AI module ecosystem. It provides comprehensive AI-powered text analysis for Drupal content, measuring various aspects including trust, objectivity, audience targeting, and reading levels.

Features

  • Configurable sentiment analysis:
    • Add/remove and order sentiments through UI
    • Enable/disable analyzer per content type
    • Enable/disable specific sentiments per content type
  • Pre-configured sentiment dimensions including:
    • Trust & Credibility (from Promotional to Authoritative)
    • Objectivity & Bias (from Opinion-Based to Fact-Based)
    • Audience Vibe Check (from Gen Z to Boomer)
    • CEFR Reading Level (from A1 Beginner to C2 Proficient)
  • Visual gauge displays for easy interpretation
  • Instant AI feedback for content creators
  • Integration with the Analyze framework

Post-installation

  1. Configure AI provider settings at /admin/config/ai/providers
  2. Configure and order sentiments at /admin/config/analyze/sentiment
  3. Enable/disable the analyzer per content type at /admin/config/content/analyze-settings
  4. Enable/disable specific sentiments per content type at /admin/structure/types/manage/{content-type}
  5. Configure permissions at /admin/people/permissions#module-analyze_ai_sentiment
  6. Access sentiment analysis through the Analyze tab on content

Additional requirements

This module requires:

  • Analyze module
  • AI module with a configured provider
  • Views Color Scales module (provides colored tables for report page)
